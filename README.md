## Technologies Used
- **MongoDB** (Database)
- **Express.js** (Backend Framework)
- **React.js** (Frontend UI)
- **Node.js** (Server)
- **JWT** (Authentication)
- **REST API**

## Features
- Complete User Registration & Login securely handled with JWT Auth.
- Personal Dashboard displaying the intern's details.
- Full Todo Application (Add, Update, Delete, Filter, Prioritize).
- Beautiful custom pure CSS styling without heavy frameworks.

## How to Run Locally

### 1. Backend Setup
1. Open a terminal in the `backend` folder.
2. Run `npm install`.
3. Make sure MongoDB is running locally.
4. Run `npm run dev` or `node server.js` to start backend on `localhost:5000`.

### 2. Frontend Setup
1. Open a terminal in the `frontend` folder.
2. Run `npm install`.
3. Run `npm run dev` to start the frontend.
4. Open the displayed local URL in your browser.
